# Delta Robot with Damped Articulations  
### A Complete Mechanical, Electrical, and Control System — with an Alternative Mathematical Language

> "This and everything inside is free to use and share and must remain forever like this."

---

## Overview

This repository contains my undergraduate thesis:

**“Construcción de un prototipo de robot Delta con articulaciones amortiguadas”**  
(University of Quindío, 2017)

But this is not just a robotics project.

This work presents a **complete system**:

- Mechanical design of a Delta robot
- Electrical architecture and actuation systems
- Full kinematic and dynamic modeling
- Optimal control (LQR, Riccati, Luenberger observer)
- Experimental implementation

And something more:

> A **custom mathematical language** designed to describe real physical systems in a unified way.

---

## Why this matters

Most works separate:

- Geometry  
- Dynamics  
- Control  

This work attempts to **unify them** under a single symbolic structure.

The notation used here is unconventional on purpose:

- It treats vectors, tensors, and systems as **coherent entities**
- It prioritizes **interaction over abstraction**
- It is built to describe **real systems**, not just ideal ones

This language was never formally published or validated by academia.

I am publishing it now because:

> I believe it contains something valuable.

## Key ideas

- A Delta robot can be understood as a **set of interacting constraints**
- Control emerges from **state relationships**, not isolated equations
- Real systems do not require perfect models — only **closed feedback loops**
- Mathematics should adapt to reality, not the other way around
